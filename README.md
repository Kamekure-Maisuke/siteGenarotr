# Docusaurus試用リポ
## ファーストプロジェクト手順

- 下記のコマンドで、Docusaurusの導入とチュートリアルページ作成を行う。

```bash
# Dcousaurusの導入
$ npm install --global docusaurus-init
# プロジェクトフォルダの作成と移動
$ mkdir ~/work/docusaurus-sample
$ cd ~/work/docusaurus-sample
# Docusaurusプロジェクトの雛形作成
$ docusaurus-init
# 公開ページフォルダへの移動
$ cd website
# ローカルサーバーの起動
$ npm start
```

- `localhost:3000`へアクセスして、下記の画面が表示されたら、完了。
